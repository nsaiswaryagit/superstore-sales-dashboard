# Superstore Sales Dashboard

## Overview
Interactive sales dashboard built using Google Sheets on Superstore dataset (10,195 rows).

## Key Insights
- High discounts reduce profitability
- Some sub-categories generate losses
- Sales trend increasing year over year

## Tools Used
- Google Sheets
- Pivot Tables
- Data Visualization
